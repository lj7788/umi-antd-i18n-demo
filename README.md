# umi-antd-i18n-demo
React基于umi+dva+antd+i18n工程模板

# 技术栈
1. React
2. umi
3. dva
4. antd
5. react-intl-universal
6. 服务端使用 https://github.com/lj7788/sequelize-express-mysql-demo

# 说明
1. 自定义页面可以继承MyPage,就会有内置方法i18n，用于国际化处理
